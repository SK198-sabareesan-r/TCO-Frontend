# Running the Frontend

## Prerequisites
- Node.js installed
- Navigate to the project directory:
  ```
  cd "c:\Users\Dell\Desktop\AWS_Migration_Frontend\AWS Frontend\cloud-migration-ui"
  ```

## Install Dependencies
Run this once before starting the app:
```
npm install
```

## Development
```
npm run dev
```
Open `http://localhost:3000` in your browser.

## Production
```
npm run build
npm run start
```
The app will be served at `https://tco.shellkode.ai`.
